# FileConverter

A Python desktop application for converting documents, images, audio/video, archives, and 3D model files.

## Features

- User accounts
- File conversion
- Local file library
- Batch conversion (up to 100 files)
- Subscription system
- Multiple output formats

## Technologies

- Python
- Tkinter
- SQLite
- Pillow
- FFmpeg
- Pandas

## Installation

pip install -r requirements.txt

python3 fileconverter.py
